# SPA Usuarios - Vue 3 + Vuetify

## 🛠️ Instalación

```bash
npm install
npm run dev
```

## 💡 Tecnologías

- **Vue 3** (Composition API)
- **Vuetify 3**
- **Axios**

## ⚙️ Funcionalidad

- Listado de usuarios desde API pública.
- Modal con detalles.
- Filtro en tiempo real.
- Responsive con Vuetify.
- Carga, manejo de errores y animaciones.
